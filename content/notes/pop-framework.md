I find that using this framework when reporting my progress during standup meetings is very concise and helpful. Before this, I usually struggled and thought a lot about what I should say, how long or how short should I report. My manager taught me this framework and definitely it helps my report more concise, and helps team members to grab the context quickly.

POP - standing for purpose - outcome - progress

Purpose: Why do you do that task?

Outcome: How do you expect the outcome look like?

Progress: How much has you done on this?

For example:
Today, I will continue working on analytics report. 
(P) This report is very informative for sale team to understand customer's reactions regards our new campaigns, which will help them measure the success and improve for the next one. 
(O) I expect to deliver a dashboard and a 2-page report about the insights.
(P) I've done with the draft visualizations and will add more comments on each chart to finish the report today.
